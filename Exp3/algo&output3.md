Experiment – 03

\# Write a program to implement string functions

**Algorithm**



STEP1: Start the program.



STEP2: Ask the user to enter a string.



STEP3: Use the len() function to find the length of the string and display it.



STEP4: Ask the user to enter another string.



STEP5: Convert the string to uppercase using the upper() function and display it.



STEP6: Ask the user to enter another string.



STEP7: Convert the string to lowercase using the lower() function and display it.



STEP8: Ask the user to enter another string.



STEP9: Reverse the string using slicing \[::-1] and display the reversed string.



STEP10: Ask the user to enter two strings.



STEP11: Concatenate the two strings using the + operator and display the result.



STEP12: Ask the user to enter two strings again.



STEP13: Compare both strings using the equality operator ==.



STEP14: If both strings are equal, display Strings are equal, otherwise display Strings are not equal.



STEP15: Ask the user to enter a main string and a substring.



STEP16: Check whether the substring exists in the main string using the in operator.



STEP17: If present, display Substring found, otherwise display Substring not found.



STEP18: Stop the program.



**Output**

Length of the string: 5

Enter a string: PROGRAMMING

Enter a string: apple

Enter first string: good

Enter second string: morning

Concatenated string: good morning

Strings are equal
